# 🪙 Artha Tracker

A simple mobile app to help users track their personal budget and finances.

## Main Features
- Login page with form validation
- Budget & expense overview
- Transaction categories
- Recent transaction history

## Tech Stack
- Flutter (Dart)
- Google Fonts

## How to Run
```bash
flutter pub get
flutter run
```
